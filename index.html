Головні команди Git

git clone - клонування репозиторію;
git status - перевірка стана файлів репозиторія;
git add - додавання під версіоний контроль нових файлів;
git commit - фіксація змін в репозторії;
git branch - створення нової гілки у репозиторії;
git merge - злиття двох гілок;
git pull - отримання змін у віддаленій гілки репозиторія і злиття зі своїми змінами в поточній клонованій;
git push - відправдення змін в віддалений репозиторій.
